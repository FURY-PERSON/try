# Инструкция: генерация 10 000 фактов через Claude Code

## Структура

- Результаты сохраняются в `generated-facts/`
- Каждый батч — отдельный файл: `batch-001.json`, `batch-002.json`, ...
- Финальный объединённый файл: `generated-facts/all-facts.json`

## Шаг 1. Генерация батчами

Открой Claude Code и вставляй команды ниже одну за другой.
Каждая команда генерирует 50 фактов для конкретной категории и сложности.

**Всего: 200 батчей × 50 = 10 000 фактов**

### Матрица батчей

10 категорий × 5 difficulty × 4 батча = 200 батчей

| # батчей | Категория   | Difficulty | Подтемы для разнообразия                    |
|----------|-------------|------------|---------------------------------------------|
| 1-4      | science     | 1-5        | физика, химия, биология, математика         |
| 5-8      | history     | 1-5        | древний мир, средневековье, новое время, XX век |
| 9-12     | geography   | 1-5        | страны, реки/горы, города, демография       |
| 13-16    | nature      | 1-5        | животные, растения, океан, экосистемы       |
| 17-20    | technology  | 1-5        | IT, изобретения, транспорт, энергетика      |
| 21-24    | culture     | 1-5        | искусство, музыка, кино, литература         |
| 25-28    | sports      | 1-5        | олимпиада, футбол, рекорды, необычные виды  |
| 29-32    | health      | 1-5        | тело, питание, медицина, психология         |
| 33-36    | space       | 1-5        | солнечная система, звёзды, космонавтика, экзопланеты |
| 37-40    | food        | 1-5        | история еды, кухни мира, напитки, продукты  |

## Шаг 2. Промпт-шаблон

Копируй и меняй параметры `[CATEGORY]`, `[DIFFICULTY]`, `[SUBTOPICS]`, `[BATCH_NUM]`:

---

Прочитай файл `facts.md` — в нём описаны все критерии качества, формат JSON и примеры хороших/плохих утверждений. Строго следуй всем правилам из этого файла.

Сгенерируй 50 утверждений (≈25 фактов и ≈25 фейков) и сохрани в файл `generated-facts/batch-[BATCH_NUM].json`.

Параметры:
- Категория: [CATEGORY]
- Сложность: [DIFFICULTY] из 5
- Подтемы для разнообразия: [SUBTOPICS]

Сохрани результат как валидный JSON-массив в файл. Только JSON, без обёрток.

## Шаг 4. Второй проход (×2)

После первого прохода (100 батчей = 5000 фактов) — делаешь второй проход с другими подтемами.
Батчи 101-200 — те же категории и сложности, но с пометкой:

> ВАЖНО: Вот список тем, которые уже были покрыты в первом проходе: [вставить список из первого прохода]. Генерируй ТОЛЬКО новые темы, не пересекающиеся с уже существующими.

## Шаг 5. Объединение

После генерации всех батчей, попроси Claude Code:

> Объедини все файлы из `generated-facts/batch-*.json` в один файл `generated-facts/all-facts.json`. Проверь что все элементы валидный JSON, убери дубликаты по полю `statement`, выведи статистику: общее количество, распределение по категориям и сложностям, соотношение фактов/фейков.

## Шаг 6. Валидация URL

> Напиши и запусти скрипт, который проверит все `sourceUrl` и `sourceUrlEn` из `generated-facts/all-facts.json` на доступность (HTTP HEAD request). Сохрани отчёт о битых ссылках в `generated-facts/broken-urls.json`.

## Советы

- Если Claude Code начал повторяться — укажи конкретнее подтемы
- Если качество падает — уменьши батч до 30 штук
- Можно запускать несколько сессий Claude Code параллельно для разных категорий
- Периодически проверяй файлы — если JSON невалидный, попроси пересгенерировать батч
